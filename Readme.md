## How To Use

### **Installation**

#### **SDL2 Library**

```console
$ sudo apt-get install libsdl2-dev
```

##### **Additional SDL2 Libraries :**

- ##### SDL Image

```console
$ sudo apt-get install libsdl2-image-dev
```

- ##### SDL Font

```console
$ sudo apt-get install libsdl2-ttf-dev
```

- ##### SDL Mixer

```console
$ sudo apt-get install libsdl2-mixer-2.0-0
```

### **Build and Run**

```console
$ make && ./run
```
